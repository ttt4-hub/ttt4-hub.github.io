Smart Triage: Patient Condition Monitoring AI

An intelligent triage system that analyzes real‑time patient vitals — such as heart rate, oxygen saturation, and blood pressure — to predict clinical urgency inside Emergency Departments. The model is engineered for high‑stakes medical environments, with dedicated handling of class imbalance, data‑integrity validation, and concept drift simulation to ensure stability when patient patterns change over time. By testing the model under feature‑shift and label‑noise drift scenarios, the system demonstrates how reliably it can prioritize life‑threatening cases even when real‑world hospital data evolves.

Tourist Trip Planner

A C++ and Python application that helps users plan customized travel routes between cities in Europe and the Middle East. It models cities and distances using graph data structures and applies Dijkstra’s algorithm for intelligent pathfinding. Users can choose transportation modes (car, train, airplane), set trip preferences (Nature, History, Culture, Adventure, Relaxation), and collaborate on shared travel plans. The system supports cost and fuel estimation, CSV‑based save/load functionality, and user feedback collection.
